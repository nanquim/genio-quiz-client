# Gênio Quiz

Front-end em VueJS para o servidor ExpressJS

#### https://genio-client.herokuapp.com/#/

## 1. Instale o projeto
```
npm install
```

### Executar em desenvolvimento
```
npm run serve
```

### Compila e minifica para produção
```
npm run build
```

### Linter
```
npm run lint
```

### Executar em produção
```
npm run start
```
